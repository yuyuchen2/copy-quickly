# 🚀 Copy Quickly

一个强大的油猴脚本，让复制粘贴变得超快速！

## ✨ 功能特性

### 1. **快速粘贴** 📋
- ✅ 对所有网页有效
- 当你在输入框/文本框输入内容时，自动在旁边显示 **"粘贴"** 按钮
- 点击按钮即可快速将剪贴板内容粘贴到输入框
- 支持所有类型的输入框（input、textarea）

### 2. **快速复制** 📄
- ✅ 对所有网页有效
- 选中任何文本时，自动显示 **"复制"** 按钮在附近
- 点击按钮一键复制到剪贴板
- 支持各种网页内容（文本、链接等）

### 3. **友好的用户界面** 🎨
- 🟢 绿色粘贴按钮（输入框旁）
- 🔵 蓝色复制按钮（选中文本旁）
- 平滑的悬停动画效果
- 自动调整位置，确保不超出视图
- 中文文本提示，操作反馈清晰

## 📦 安装方法

### 前置要求
- 浏览器已安装 **Tampermonkey** 扩展
  - Chrome: [Tampermonkey on Chrome Web Store](https://chrome.google.com/webstore)
  - Firefox: [Tampermonkey on Firefox Add-ons](https://addons.mozilla.org/firefox/)
  - Edge/Safari: 搜索 Tampermonkey

### 安装步骤

1. **打开脚本文件**
   - 访问本仓库的 `copy-quickly.user.js` 文件
   - 或者直接点击 [这个链接](https://github.com/yuyuchen2/copy-quickly/raw/main/copy-quickly.user.js)

2. **安装脚本**
   - Tampermonkey 会自动弹出安装窗口
   - 点击 **"安装"** 按钮确认
   - 或者手动复制脚本内容到 Tampermonkey 管理界面

3. **启用脚本**
   - 在 Tampermonkey 管理界面确保脚本处于启用状态
   - 刷新网页，脚本生效

## 🎯 使用方法

### 粘贴操作

```
1. 点击任何网页上的输入框/文本框
2. 开始输入内容
3. 在输入框旁边会出现 "📋 粘贴" 绿色按钮
4. 点击按钮，剪贴板中的内容会自动填充到输入框
```

### 复制操作

```
1. 在网页上选中任何文本
2. 在选中文本附近会出现 "📄 复制" 蓝色按钮
3. 点击按钮，选中的文本会自动复制到剪贴板
4. 按钮显示 "✓ 已复制" 表示复制成功
```

## 🔧 功能详情

### 智能特性
- 🎯 **自动检测输入框** - 支持 `<input>` 和 `<textarea>` 元素
- 🌐 **全网兼容** - 对所有网站有效
- 🔄 **动态内容支持** - 自动监听新加载的输入框
- 📍 **智能定位** - 按钮自动避开屏幕边界
- 💬 **中文界面** - 完全中文化设计
- ⚡ **性能优化** - 轻量级脚本，不影响网页速度

### 按钮行为
| 操作 | 按钮颜色 | 文本 | 位置 |
|------|---------|------|------|
| 粘贴 | 🟢 绿色 | 📋 粘贴 | 输入框右侧/左侧 |
| 复制 | 🔵 蓝色 | 📄 复制 | 选中文本附近 |

### 反馈提示
- ✓ 成功：按钮显示 "✓ 粘贴成功" / "✓ 已复制"
- ✗ 失败：按钮显示 "✗ 粘贴失败" / "✗ 复制失败"
- 2秒后自动恢复原按钮文本

## 🐛 故障排查

### 问题：按钮没有出现
**解决方案：**
- 确认 Tampermonkey 扩展已安装并启用
- 检查脚本在 Tampermonkey 管理界面是否启用
- 尝试刷新网页
- 打开浏览器控制台（F12）查看是否有错误信息

### 问题：粘贴/复制不工作
**解决方案：**
- 确保浏览器允许网页访问剪贴板权限
- 某些网站可能有安全限制，无法访问剪贴板
- 尝试在不同网站测试

### 问题：按钮位置不对
**解决方案：**
- 这是正常现象，脚本会自动调整位置避开屏幕边界
- 如果仍有问题，可以在 Issues 中反馈
